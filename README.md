## Engineering Principles

| Principle | What It Means |
|:--|:--|
| Automation first | Repetitive manual work should become a dependable system. |
| Production-minded | Code should be deployable, observable, recoverable, and documented. |
| Simple interfaces | Good systems expose clear APIs, predictable data flow, and readable boundaries. |
| Maintainable speed | Fast delivery matters, but not at the cost of future development. |
